###基础7
```
忘记root密码
   重启系统
   在等待界面按任意键 然后 按e 选择第2项kernel/vmlinuz.....
   在按e 进入编辑模式 输入空格和1或者s（进入1级别） 回车 再次回到选择界面 按b重新引导系统
   当系统运行到可以进行命令操作的时候 输入 passwd 重新设置密码
   重设置完密码后， 输入 exit退出用户，重新启动
```

###基础7


###文件结构
``` 
ls -l 查看当前目录的时候
文件开头          -：表示普通文件 
                  d：表示目录 
                  b：设备文件（块设备，可以存储东西的设备，例如U盘等）
                  c：设配文件（字符文件，例如鼠标等）
                  l：软连接（快捷方式）

后面的9个字符     ---       ---       ---
                 用户       组         其它
                   rwx       rwx       rwx
                 r 4 读     w 2 写    x 1 执行
```

###目录结构
```
bin        : 存放2进制文件，例如命令
boot       : 系统启动文件和内核
dev        : 设备相关，U盘等
etc        : 配置文件
home       : 各个用户的目录（除了root）
lib        : 链接库文件
lost+found : 
media      : 挂载点，U盘等
mnt        ：挂载点
net        : 
opt        : 源码软件安装路径
proc       : 内核参数（没有文件，虚文件）
root       : root用户的家目录
sbin       : 2进制文件，系统管理员的命令存放
tmp        : 临时文件
user       : 软件的默认安装路径
var        : 日志，缓存的

```
